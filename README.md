# Deskting Project

Deskting project wanna be part of Classting

### Run
1. Clone git project => <br>
  `# git clone https://github.com/Deory/deskting.git`
2. Install dependencies => <br>
  `# yarn add`
3. Start deskting => <br>
  `# yarn run start`

### OpenAPI Specification
After run, you can check it on below link
